# goodechilde/gc-tags

Our common contact fields for people and places

Changes between versions will be tracked in the [CHANGELOG](CHANGELOG.md).

## Installation

```bash
composer require goodechilde/gc-tags
```
```bash
php artisan vendor:publish --tag=gc-tags 
```
